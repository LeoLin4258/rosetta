## Rosetta 0.1.0-beta.12

本次更新修复中国大陆网络环境下 PDF 首次预热可能一直卡住的问题，并重新整理 PDF 组件安装入口。

### PDF 预热不再依赖 Hugging Face

- PDF 组件现在内置 DocLayout-YOLO 版面模型，首次预热不会再临时访问 Hugging Face 下载权重
- 旧版 PDF 组件会被识别为需要更新，重新安装后即可使用新的自包含组件包
- PDF 组件下载增加 GitHub 镜像 fallback，改善中国大陆用户下载成功率

### PDF 组件安装更清晰

- PDF 组件统一在设置页安装、重新安装或从本地安装包导入
- 工作台不再在点击翻译时自动下载 PDF 组件，缺少组件时翻译按钮会禁用并提示先安装
- 设置页的 PDF 组件区域改为直接展示安装入口、代理配置和技术信息

### 状态提示更克制

- Header 中的 PDF 组件状态只在当前打开 PDF 文件时显示
- PDF 组件缺失或需要更新时，Header 提示可直接跳转到设置页
- 修复 PDF 状态提示的换行和溢出问题，长模型路径不再挤出 tooltip
