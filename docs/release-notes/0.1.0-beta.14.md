# Rosetta 0.1.0-beta.14

Rosetta now ships on Windows x64 and macOS Apple Silicon from one shared
release baseline.

## Highlights

- First Windows release for Windows 10/11 x64.
- Managed local translation on supported NVIDIA GPUs (SM75 or newer).
- More reliable first-run installation with concrete errors and retry support.
- Clear PDF verification, extraction, installation, and warmup progress.
- Translation waits for the managed runtime to become ready.
- New installations follow the operating system light or dark appearance.
- Windows exits and cleans up managed processes when the main window closes.
- macOS keeps its native close-to-hide and Dock restore behavior.

## Supported platforms

- Windows 10/11 x64. The managed local model requires NVIDIA SM75 or newer.
  Other Windows machines can use an explicitly configured external API.
- macOS 13 or newer on Apple Silicon.

Rosetta remains local-first. Documents, translations, job caches, prompts, and
runtime logs are not uploaded by the release or update system.
