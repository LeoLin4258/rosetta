## Rosetta 0.1.0-beta.13

本次更新优化 PDF 引擎预热的反馈，以及 PDF 译文预览在窗口缩放时的体验。

### PDF 引擎预热反馈更清晰

- 预热过程按阶段显示进度：「加载 PyTorch」「加载文档版面模型库」「加载 pdf2zh」「完成准备」，每个阶段同时显示已用秒数
- 首次安装或升级后 30 秒级的冷启动不再像卡死，秒数本地每秒更新
- 预热过程中工作台的翻译按钮显示为禁用状态，不再单独显示重复的预热进度

### PDF 预览不再随窗口闪烁

- 译文页面统一按最高清晰度渲染一次，调整窗口、切换侧栏、半屏吸附不再重新生成图片
- 调整窗口大小时通过 CSS 缩放，PNG 字节不变，体验更平滑

### PDF 源文与译文严格等宽

- 之前源文左侧的页面勾选框占用宽度导致源文图片比译文窄，两边页面尺寸不一致
- 现在两边都保留相同宽度的控件栏（译文侧为空占位），同一页的源文与译文图片严格等宽等高
